# Currículo Responsivo (HTML + CSS)

Este projeto consiste no desenvolvimento de um currículo online utilizando apenas **HTML5** e **CSS3**, como atividade da disciplina de Desenvolvimento Web.

O objetivo foi criar uma página estática, organizada e responsiva, aplicando conceitos fundamentais de estruturação e estilização de páginas web.

---

## 🚀 Tecnologias Utilizadas

- HTML5  
- CSS3 (Mobile First)

---

## 📱 Responsividade

O projeto foi desenvolvido seguindo a abordagem **mobile-first**, garantindo adaptação para diferentes tamanhos de tela:

- **Mobile (até 767px)**  
  Layout em coluna única, com todos os elementos empilhados verticalmente.

- **Tablet (768px a 1023px)**  
  Layout semelhante ao desktop, com organização em duas colunas.

- **Desktop (a partir de 1024px)**  
  Layout em duas colunas, com sidebar (aside) ocupando menos espaço e o conteúdo principal em destaque.
